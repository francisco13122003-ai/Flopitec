# Flopitec Servicios Informáticos — Web pública V1

Proyecto web público corporativo independiente para Flopitec Servicios Informáticos. Es una primera entrega estática, revisable y preparada para GitHub + Cloudflare Pages.

## Stack
- Astro
- Tailwind CSS
- TypeScript
- Web estática
- SEO técnico básico con sitemap, robots, canonical, Open Graph y JSON-LD

## Comandos
```bash
npm install
npm run dev
npm run build
npm run preview
```

Cloudflare Pages futuro: build command `npm run build`, output directory `dist`.
